# CSS Box Shadow Generator

A visual CSS box-shadow generator: set offset, blur, spread and colour with sliders, preview live, and copy the CSS. Zero dependencies — works in plain HTML, React, Vue, Svelte or Astro.

**▶ [Live demo](https://sgbp.tech/tools/box-shadow-generator)**

```html
<script src="box-shadow.js"></script>
<box-shadow-generator></box-shadow-generator>
```

## What it does

The CSS box-shadow property casts a shadow around an element from an x/y offset, a blur radius, a spread radius and a colour — and can be inset for inner shadows.

## Install

```bash
npm install @sgbp/box-shadow
```

or copy `box-shadow.js` into your project.
